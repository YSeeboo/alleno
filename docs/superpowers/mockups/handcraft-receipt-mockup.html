本次更新（对应你的反馈）

  1. 配件回收回执条常驻：无论是否打开弹窗，主表上方始终显示「配件回收 · N 项」摘要，可点编辑。
  2. 回执编码查询：填入编码 → 商家自动带出并锁定、列表仅显示该单产出项，并有蓝色 scope 提示条 + 「✕ 清除」回到商家模式；支持不选商家、只填编码直接查询。
  3. 手工单独立成列：从产出项编号行里移除，单独作为金额之后的最后一列，HC-xxxx 🗗 为可点链接，点击打开手工单详情弹窗（只读预览 + 「打开完整手工单」）。移动端则作为卡片副信息行内的可点链接。
